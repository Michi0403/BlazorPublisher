# PublisherStudio v1.0.10 beta polish

- Reorganized the Publisher, Picture Studio, Media Studio, Story Editor, and Barcode Studio commands into categorized DevExpress ribbon menus.
- Restored readable text for primary ribbon commands such as Insert Video, Barcode / QR, DOCX, and Update Fields.
- Added snap-to-object movement with temporary green alignment, orange proximity, and red collision feedback.
- Added shape conversion commands for rectangle, rounded rectangle, ellipse, and line objects.
- Hardened direct media insertion and Media Studio imports with browser-file inspection plus a ranged local-media fallback.
- Isolated printing from open File/export menus and application overlays.
- Changed PNG/JPEG page export to prefer the browser's working SVG renderer and reject blank raster results before falling back.
- Fixed QR square, rounded, and dot modules; QR correction metadata; barcode transparency; and valid defaults for Code 39, EAN-13, UPC-A, ITF-14, and Codabar.
