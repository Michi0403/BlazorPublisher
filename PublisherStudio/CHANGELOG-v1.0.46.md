# PublisherStudio 1.0.46

- Added a self-updating **Publication media** data object with one row per picture, video, or audio element.
- Gallery and Tile View now render mixed image, video, and audio rows through explicit media field mappings.
- Added a one-click **Use pictures & videos from document** workflow in Component Studio.
- Added DevExtreme **Chat** to Component Studio and the Insert ribbon.
- Chat rows support platform, channel, message, timestamp, author, and avatar mappings.
- Added a strict platform-isolated chat bridge contract for Twitch, YouTube, custom adapters, and local preview.
- Added platform/channel runtime overrides for separate stream-render variants without mixing messages.
- Added modern media cards, video/audio controls, empty states, and chat styling.
