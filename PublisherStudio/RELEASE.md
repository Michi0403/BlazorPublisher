# PublisherStudio v1.0.49 release

See `CHANGELOG-v1.0.49.md`, `docs/STREAMING.md`, `docs/COMPONENT_RUNTIME.md`, and `VALIDATION.md`.

This revision aligns Streaming Studio with the PublishingSuite/SubApp interface, adds contextual ribbon and right-click workflows, restores double-click and context-menu activation for embedded canvas components, and introduces application-wide explanatory tooltips while preserving the existing publication and Media Host wiring.

The source package intentionally excludes `bin`, `obj`, `.vs`, `.git`, restored `node_modules`, local credentials, recordings, and generated runtime caches.

Application/package/installer/Media Host `1.0.49`, publication format `1.45`, picture format `1.2`.
